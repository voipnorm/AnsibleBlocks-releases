# AnsibleBlocks — Releases

This repository hosts release artifacts for **AnsibleBlocks**, a visual Ansible playbook builder.

## Downloads

Head to the [**Releases**](https://github.com/voipnorm/AnsibleBlocks-releases/releases) page to download the latest version for your platform:

| Platform | Format |
|----------|--------|
| macOS (Universal) | `.dmg` / `.zip` |
| Windows (x64) | `.exe` (installer) / `.exe` (portable) |

## Auto-Updates

If you already have AnsibleBlocks installed, the app checks for updates automatically and will notify you when a new version is available. You can also check manually from **Settings → Updates → Check Now**.

## Source Code

The source code for AnsibleBlocks is maintained in a separate private repository. This repo is used exclusively for distributing release binaries.
